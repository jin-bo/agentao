# 专业高级感 · 高端商务演示文稿制作指南

## 总体原则

- 以**顶级商业编辑（Editorial Premium）**与**高端咨询（McKinsey / BCG / 招股书 / Apple Keynote）**的视觉语言诠释演讲内容
- 每页图像须忠实传达该页核心信息；构图克制、留白充足、信息分层清晰
- 全稿风格高度一致；情节转折时仅通过**色彩饱和度**与**光照角度**的细微变化体现，不破坏统一感
- 不要出现 血腥、暴力、武器 相关的文字与画面
- 画面**图文结合**，可直接用于现场演讲展示，文字必须以排版关系融入构图，**禁止悬浮孤立**

### 风格原创合规原则

**严禁在提示词中使用任何受版权保护的品牌名、产品名或专有图形**，包括但不限于：
- ❌ 苹果、Apple、iPhone、Tesla、Bloomberg、McKinsey、Goldman 等具体品牌或其标识
- ❌ 任何受版权保护的字体名（如 SF Pro、Helvetica 商业版）的精确再现

**用以下方式描述风格意图（替代品牌名）**：
- ❌ "Apple Keynote 风格" → ✅ "极简高端发布会幻灯片风格、巨幅留白、精准对齐、哑光金属质感"
- ❌ "McKinsey 风格" → ✅ "顶级管理咨询报告风格、严密信息层级、克制配色、几何化数据可视化"
- ❌ "Bloomberg Terminal 风格" → ✅ "金融终端编辑风格、深色背景、金色高光强调关键指标"

## 视觉语言系统（必读 · 全稿统一）

本指南定义的「专业高级感」由四个固定维度构成。**所有页面都必须严格遵循下述系统**，否则会失去统一性与高端感。

### 1. 色彩系统（Color System）

| 角色 | 名称 | 色值 | 用途 |
|------|------|------|------|
| **主色调** | 极浅灰（Pearl Mist） | `#F2F2F2` | 主要文字、信息卡片背景、关键数据数字、图表主体、留白质感 |
| **辅助色 A** | 金属灰（Brushed Metal） | `#8A8F98` ~ `#B8BCC2`（哑光金属梯度） | 次级文字、分割线、几何结构、装饰性几何体、冷光质感 |
| **辅助色 B** | 高级金（Champagne Gold） | `#C9A961` / `#D4B57A`（哑光香槟金，非俗气黄金） | 关键指标高亮、重点数据、点睛装饰、获奖/成就标识、勋章质感 |
| **背景色** | 深蓝灰（Midnight Navy） | `#0F233B` | 主背景，几乎所有页面都使用，营造深邃专业氛围 |

**配色铁律：**
- 任意一页画面：深蓝灰背景占 60-75%；浅灰元素占 20-30%；金属灰占 5-15%；高级金 ≤ 5%（仅作点睛）
- **金色不可大面积使用**——只用于关键数字、徽章、细线点缀，否则会显得俗气而非高级
- 浅灰文字必须与深蓝灰背景产生足够对比，可在浅灰下加 5%-10% 不透明度的金属灰渐变以避免「死白」
- 严禁出现：高饱和的红、绿、蓝、紫等彩色 RGB 原色；卡通配色；荧光色；暖橙色

### 2. 光照与材质（Lighting & Material）

- **主光源**：来自画面斜上方的柔和顶光，模拟摄影棚布光（Softbox），在金属表面形成温润高光
- **次光源**：背景轻微的体积光（Volumetric Light）/ 雾化光晕，营造空间纵深感
- **材质语言**：
  - 哑光金属（Brushed Aluminum）质感的几何体与文字
  - 香槟金的窄边框 / 细线（hairline）
  - 半透明毛玻璃（Frosted Glass）信息卡片
  - 深蓝灰背景带极轻微的颗粒感（Film Grain），避免数字化的廉价平面感
- **禁止**：塑料反光、卡通描边、廉价 3D、过曝高光、廉价镭射效果

### 3. 排版与版式（Typography & Layout）

- **网格系统**：采用 12 列基础网格，左右各 1 列空白边距，所有元素**严格对齐基线**
- **字体语言**（在提示词中描述质感，不指定具体商业字体）：
  - 标题：极细 / 中粗的现代无衬线字体（Modern Sans-Serif），字距宽松（letter-spacing 较大），呈现「编辑大字报」气质
  - 关键数据：超粗的 Display 字体或细衬线（Didone 风格的 thin serif），用于「巨型数字」呈现
  - 正文与图注：极细无衬线，字号克制
- **版式偏好**：
  - 大量留白（白银比例 / 黄金比例）
  - 标题与内容之间常用一条细金色 hairline 分隔
  - 信息以「卡片 / 模块 / 表格」形式呈现，而非段落散文
  - 角标（如 `01 / 12`、`§ 02`、`CHAPTER ONE`）出现在画面左上或右下，强化「报告页」气质

### 4. 视觉构图（Composition）

- 单页画面以**单一视觉焦点**为主（一个 Hero 元素 + 周边卫星信息）
- 几何化、对称或黄金分割构图，避免拥挤
- 镜头视角：以**正面视角**或**轻微等距 30° 透视（Isometric-like）**为主，避免过度倾斜
- 镜头景深：浅景深（Shallow DoF）用于强调焦点元素

## Step 1：理解讲稿内容与受众

在生成任何提示词之前，先完整阅读用户提供的讲稿，提取：

| 维度 | 提取内容 |
|------|----------|
| **主题** | 讲稿核心议题与关键论点 |
| **受众** | 面向谁（董事会 / 投资人 / 行业峰会 / 高管 / 媒体发布会） |
| **目的** | 战略汇报 / 融资路演 / 产品发布 / 年度回顾 / 行业洞察 |
| **情感基调** | 严谨克制 / 自信沉稳 / 远见卓识 / 庄重权威 |
| **内容结构** | 页数、各页标题与核心要点 |
| **核心数据** | **每页是否有可被「巨型化」呈现的关键数字 / 比率 / 排名** |
| **核心隐喻** | **为抽象概念（如增长、信任、生态、转型）寻找几何化或物体化的视觉象征** |

### 1.1 情感旅程弧度

将全稿的情感节奏画成一条克制的曲线，例如：

```
封面（庄重期待）→ 行业背景（沉稳客观）→ 挑战 / 痛点（凝重严谨）→ 战略 / 解决方案（笃定自信）→ 数据成果（高光时刻）→ 总结展望（远见卓识）
```

「专业高级感」的情感弧度幅度**远小于**戏剧化风格，所有变化都通过**金色出现的多寡、光线方向、留白密度**等微妙手法实现，**不要靠夸张表情或饱和色彩**来推动情绪。

### 1.2 每页核心视觉时刻

对每一页，提炼出**一句话**：「这一页最应该让观众感受到什么？」

> 示例：
> - 封面：「这是一份足够分量的发言，发布方有充分准备」
> - 数据页：「这个数字本身就是结论，无需多言」
> - 战略页：「方案严密、各模块各司其职、整体可信」
> - 总结页：「眼光放远、格局已成」

这句话决定了 Hero 元素、留白比例、是否使用金色点睛。**不能用主题词代替**（「讲战略转型」不是视觉时刻）。

## Step 2：生成高级商务视觉提示词

### 2.1 整体风格与「视觉资产」设定（frontmatter `style` + `characters` 字段）

在 YAML frontmatter 中写入贯穿全稿的**风格描述**和**视觉资产设定**。

#### `style` 字段 — 全局风格

```yaml
style: |
  以顶级商业编辑（Editorial Premium）与高端管理咨询报告的视觉语言创作演示文稿单页画面。
  深蓝灰背景（#0F233B）占据画面主体，极浅灰（#F2F2F2）作为信息主色，哑光金属灰用于结构与分割，
  香槟金（#C9A961）仅作关键点缀。摄影棚柔光布光，浅景深，哑光金属与毛玻璃质感，
  极轻微胶片颗粒感避免数字平面化。版式严谨对齐，留白充足，单页单一视觉焦点。
  画面图文并茂：标题以宽字距现代无衬线大字呈现，关键数据以巨型 Display 字号占据焦点，
  辅助信息以信息卡片 / 细金色 hairline 分割线 / 角标编号方式呈现，禁止文字孤立悬浮。
```

#### `characters` 字段 — 视觉资产设定 (核心一致性原则)

> **复用脚本说明**：本目录复用 `skills/zootopia-ppt/scripts/image_gen_ppt.py`。该脚本会把 frontmatter 的 `characters` 列表自动注入每页提示词。在「专业高级感」语境下，**`characters` 不再是拟人化角色**，而是「全稿反复出现的视觉资产 / 视觉锚点（Visual Anchors）」——例如标志性几何图腾、品牌色块组、装饰构件、可选的人物剪影。

为保证全稿视觉资产、几何符号、色彩组合的高度一致，**所有在文稿中反复出现的视觉锚点，必须在第一页之前就在 frontmatter 的 `characters` 列表中完整定义**。

每个视觉资产包含以下属性（沿用脚本字段名，但语义改为视觉资产）：
- `name`：资产简称（在提示词中引用时使用，如"金线徽章"、"等距塔楼"）
- `species`：资产类型（如"几何图腾"、"金属构件"、"人物剪影"、"信息图模块"）
- `appearance`：外观特征（形状、材质、尺寸、光感）
- `outfit`：表面处理 / 配色（必须落在四色系统内）
- `personality`：气质 / 出现时机（庄重发布、关键数据、章节分隔等）
- `accessories`：附属装饰（细金线、勋章纹、刻度尺、阴影投射等）

**视觉资产配置原则 (Asset-Section Alignment)**：
- **庄重型资产（如金线徽章 / 勋章构件）**：用于封面、章节分隔、获奖与成就页
- **结构型资产（如等距塔楼 / 模块矩阵 / 网格底图）**：用于战略架构、产品体系、组织架构页
- **数据型资产（如巨型数字模板 / 折线柱状装饰）**：用于成果数据、增长曲线、关键指标页
- **人文型资产（如剪影人物 / 抽象人物轮廓）**：在需要表达「客户 / 团队 / 受众」时少量出现，**禁止具像化具体人物面容**
- **隐喻型资产（如几何山脉 / 罗盘 / 阶梯）**：用于愿景、战略、转型、远见类页面

```yaml
characters:
  - name: 金线徽章
    species: 庄重型几何资产 / 圆形标识
    appearance: 直径占画面约 1/8 的圆形构件，由极细的香槟金 hairline 双圈构成，圈内为深蓝灰背景叠加纤细的星芒/罗马数字
    outfit: 香槟金细线（#C9A961）+ 深蓝灰底（#0F233B）+ 内部极浅灰文字（#F2F2F2）
    personality: 出现于封面、章节首页、总结页，赋予内容仪式感与权威感
    accessories: 底部常配一条 1px 香槟金 hairline 与角标编号（如 01 / 12）

  - name: 等距塔楼
    species: 结构型几何资产 / 等距构件
    appearance: 由若干长方体堆叠而成的极简等距投影建筑，比例修长，呈现 30° 等距视角
    outfit: 主体哑光金属灰（#8A8F98 - #B8BCC2 渐变），顶部边缘以一道香槟金薄光勾勒
    personality: 出现于「战略架构」「能力体系」「产品矩阵」类页面，象征严密结构与稳固支柱
    accessories: 底部投射柔和体积光阴影，背景可有极淡的网格参考线

  - name: 巨型数字模板
    species: 数据型版式资产
    appearance: 一个超大字号（占据画面 30%-50% 高度）的极细衬线 / 极粗无衬线 Display 数字，旁边附带克制的单位与说明
    outfit: 数字本体使用极浅灰（#F2F2F2）或哑光金属灰渐变；单位与说明使用更小字号的香槟金（#C9A961）
    personality: 出现于「关键数据」「增长成果」「市场规模」页，让数字本身成为视觉主角
    accessories: 数字下方一条细金色 hairline，以及一行极小的注释 / 数据来源

  - name: 剪影人物
    species: 人文型抽象资产 / 中性人物轮廓
    appearance: 单色剪影或半透明轮廓，体态正式（站立 / 落座），不显示具体五官，强调姿态与构图
    outfit: 极浅灰半透明剪影（#F2F2F2 @ 50% 不透明度）或哑光金属灰，绝不出现品牌服装
    personality: 出现于「客户」「团队」「受众」「愿景」等需要人文象征的页面，**少量、克制**
    accessories: 可与「等距塔楼」「金线徽章」组合出现，但绝不抢主焦点

  - name: 几何山脉
    species: 隐喻型资产 / 抽象地形
    appearance: 由数层折线 / 三角面构成的极简几何山脉，呈现远近层次
    outfit: 远景层使用半透明金属灰（雾化），近景一条山脊以香槟金细线勾勒
    personality: 出现于「愿景」「远见」「转型路径」「行业格局」类页面，象征远方目标与跨越路径
    accessories: 山顶可叠加一颗微小的金色亮点，象征目标坐标
```

**提示词中的引用方式**：在每页提示词中直接使用简称（如"金线徽章"、"等距塔楼"），脚本注入的视觉资产设定会让生图模型理解其完整形态。

> **注意**：视觉资产列表可根据讲稿内容增减。一份十页的演示稿通常只需要 3-5 个视觉资产即可贯穿始终；过多反而破坏统一感。

### 2.2 内容类型与视觉资产对应表

根据页面内容类型选择合适的视觉资产与版式：

| 内容类型 | 推荐视觉资产 / 版式 | 情感基调 |
|----------|--------------------|---------|
| 封面 / 开场 | 金线徽章居中 + 大字距标题 + 底部细金色 hairline + 角标 | 庄重、克制 |
| 章节分隔页 | 巨型章节编号（如 `02`）以哑光金属灰呈现 + 极浅灰章节标题 | 仪式、停顿 |
| 行业 / 背景 | 几何山脉远景 + 浅景深 + 角落小卡片提供数据脚注 | 沉稳、客观 |
| 痛点 / 挑战 | 等距塔楼出现裂缝 / 缺角，背景体积光略暗，金色克制 | 凝重、严谨 |
| 战略 / 框架 | 等距塔楼矩阵 + 顶部细金线连接 + 模块标签 | 严密、笃定 |
| 关键数据 | 巨型数字模板占据画面焦点 + 单位与说明 + 细金 hairline | 高光、决定性 |
| 增长 / 趋势 | 极简折线 / 柱状图，主线哑光金属灰，焦点节点金色 | 自信、上扬 |
| 团队 / 客户 | 剪影人物群组 + 等距塔楼背景 | 温润、人文 |
| 流程 / 步骤 | 横向阶段卡片，每阶段以金色编号标识 | 清晰、可执行 |
| 愿景 / 总结 | 几何山脉 + 山顶金色坐标 + 简短宣言 | 远见、卓识 |
| 致谢 / 收尾 | 大留白 + 居中金线徽章 + 极小落款 | 收束、余韵 |

### 2.2.1 镜头与视角

镜头视角直接决定专业感的「分量」，须在提示词中明确指定：

| 镜头类型 | 效果 | 适用情境 |
|----------|------|---------|
| **正面平视（Front Flat）** | 强对称、强秩序，最具仪式感 | 封面、章节页、总结页 |
| **轻微等距 30°（Isometric-like）** | 营造结构感与立体层次，仍保持克制 | 战略架构、产品体系页 |
| **微仰（Subtle Low Angle 5°-10°）** | 赋予焦点元素分量感，仍保持稳重 | 关键数据、巨型数字页 |
| **俯视等距（Top-down Isometric）** | 适合呈现整体布局与全局观 | 流程图、组织结构页 |
| **浅景深特写（Shallow DoF Macro）** | 聚焦质感，凸显金属 / 金色细节 | 徽章特写、收束页 |

> **专业感铁律**：避免**过度倾斜镜头（≥ 20°）**、**过度仰拍（≥ 30°）**、**鱼眼视角**——这些会显得戏剧化而非高级。

### 2.2.2 单页版式布局 (Premium Slide Layouts)

「专业高级感」演示稿的版式布局相对**克制收敛**，下列是高频可复用的版式类型：

#### 1. 基础版式
- **居中纪念碑式 (Centered Monument)**：单一焦点元素居中，四周大量留白；用于封面、章节页、总结页
- **左标题右数据 (Title / Data Split)**：左侧 1/3 放标题与小段说明，右侧 2/3 放巨型数字或关键图形；最高频
- **三栏并置 (Three-column Grid)**：三个对等模块横向并排，顶部一条细金 hairline 串联；用于「三大支柱 / 三步流程」
- **顶横标 + 底信息卡 (Header + Card Grid)**：顶部巨幅标题 + 下方 2x2 或 1x4 信息卡片；用于「特性矩阵 / 数据汇总」

#### 2. 高阶版式
- **全幅 Hero（Full-bleed Hero）**：单一巨型几何资产占据整页，文字以极小字号点缀；用于「视觉冲击转折页」
- **报告页样式 (Report Page Style)**：顶部细金 hairline + 顶部章节角标（如 `§ 03 / Strategy`） + 主体内容 + 底部页码与品牌字符；模拟招股书 / 年报版面
- **图表 + 注解栏 (Chart + Annotation Rail)**：左/中放图表，右侧细窄列放数据注解；用于「严谨数据页」
- **时间轴 (Editorial Timeline)**：横向时间轴，每个节点为一个细金圆点 + 上下交错的事件卡片

> **禁用版式**：杂志式拼贴、硬切的彩色色块对撞、大量重叠装饰、卡通对话框；这些都会破坏高级感。

#### 示例 A：封面页 — 「2026 年度战略发布」

* **核心视觉意图**：让数字本身成为结论，无需多言
* **版式布局**：左标题右数据（Title / Data Split），左 1/3 + 右 2/3
* **镜头与构图**：微仰 5° 正面，焦点为右侧的「28.6%」巨型数字；浅景深
* **视觉资产**：巨型数字模板（右侧）+ 底部一条细金 hairline

  - 左侧标题：宽字距现代无衬线，三行成段

  - 下方注释：极小字号金属灰，标注口径与同比基期
* **象征性元素**：数字下方一道 80px 长的香槟金 hairline，象征「成果落定」
* **细节质感**：数字右下方有一道柔和的金属高光扫过，模拟印刷压印的反光

### 2.5 提升提示词创意：从信息到视觉宣言（Enhancing Creativity）

为了让演示文稿不仅是「信息的排版」，而是具有**分量感的视觉宣言**：

#### 1. 拒绝「信息平铺」
- ❌ **平庸**：标题 + 三个圆角卡片排成一行
- ✅ **高级**：标题以巨幅大字独占顶部 1/3，中间一道金 hairline，下方仅一组关键数据，两侧大量留白

#### 2. 让数字成为图像
- 优先把「最重要的数据」做成巨型数字 Hero，标题反而退为副位
- 数字下方搭配 5 个字以内的小注（如「同比 / YoY」「截至 / as of 12.31」），克制不啰嗦

#### 3. 用几何隐喻替代图标
- ❌ **平庸**：用一个货币图标表示「营收」
- ✅ **高级**：用一组等距阶梯几何 + 金色顶部光点暗示「拾级而上」

#### 4. 在沉稳中点睛
- 每页**仅一处金色焦点**（一个数据 / 一根线 / 一个徽章），多则显俗
- 金色不要用作大色块，而是用作「细线 / 字 / 微光」

## 常见错误与注意事项

| 错误 | 正确做法 |
|------|----------|
| 每页重复写相同的风格说明 | 风格说明放在 `style` 字段，提示词聚焦版式与内容 |
| 提示词过于抽象（"一幅高级的演示页"） | 必须具体：版式名、资产引用、配色比例、字体气质、镜头视角 |
| 大面积使用金色 | 金色仅作点睛（≤ 5% 面积），用于细线 / 关键字 / 徽章；多则显俗 |
| 出现高饱和彩色（红绿蓝紫） | 严格在四色系统内（深蓝灰 / 极浅灰 / 金属灰 / 香槟金） |
| 文字孤立悬浮 | 文字须以排版关系融入：宽字距大标题、巨型数字 Hero、卡片角标 |
| 用卡通图标表达概念 | 用几何隐喻：山脉、塔楼、阶梯、轨迹线、金色坐标点 |
| 镜头过度倾斜 / 鱼眼 / 强动感 | 限于正面平视、轻等距 30°、微仰 5°-10° 等克制视角 |
| 跳过核心视觉意图 | 必须先定「这页让观众感受到什么」，再选版式与资产 |
| 视觉资产形态前后不一致 | 在 frontmatter 中一次性定义，所有页面引用简称即可 |
| 留白不足 / 信息过密 | 单页单一焦点，留白至少占画面 30%，信息分层清晰 |
