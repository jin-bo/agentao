# Agentao v0.2.7

## Highlights

**Agent Client Protocol (ACP)** тАФ Agentao can now be driven as a headless JSON-RPC agent runtime by ACP-compatible clients (e.g. Zed). Launch with `agentao --acp --stdio` or `python -m agentao --acp --stdio`.

## What's New

### ACP Runtime (~3,500 lines)
- Full stdio JSON-RPC server with methods: `initialize`, `session/new`, `session/prompt`, `session/cancel`, `session/load`
- ServerтЖТclient `session/request_permission` flow with `allow_once` / `allow_always` / `reject_once` / `reject_always`
- Stdout guard prevents stray `print()` from corrupting the NDJSON wire
- Capability advertisement: `text` + `resource_link` content blocks, stdio + SSE MCP transport

### Per-Session Working Directory Isolation
- `Agentao(working_directory=Path)` freezes memory, permissions, MCP config, skills, and all file/shell tools against that path
- Two ACP sessions in different repos see fully independent environments

### Session-Scoped MCP Injection
- `Agentao(extra_mcp_servers=...)` merges in-memory MCP configs on top of file-loaded `.agentao/mcp.json`
- ACP `session/new` accepts `mcpServers` on the wire

### CJK Retrieval Upgrade
- jieba word segmentation replaces character bigrams тАФ `"чЙИцЬмчобчРЖ"` тЖТ `{"чЙИцЬм", "чобчРЖ"}` instead of `{"чЙИцЬм", "цЬмчоб", "чобчРЖ"}`
- Custom dictionary support via `<home>/.agentao/userdict.txt`
- New inverted index avoids full-scan as memory store grows

### Other
- `python -m agentao` module entry point for environments where the console script isn't on PATH
- LLM log file gracefully falls back to `<home>/.agentao/agentao.log` when cwd is unwritable

## Bug Fixes
- **`Agentao()` no longer crashes in restricted/read-only environments** тАФ `sqlite3.OperationalError` on memory DB init now triggers the fallback path instead of propagating as unhandled

## Tests
- **336 new tests** covering ACP protocol, sessions, permissions, transport, MCP injection, and per-session cwd isolation
- Suite total: **1,035 tests** (up from 657)

## New Dependency
- `jieba >= 0.42.1`
